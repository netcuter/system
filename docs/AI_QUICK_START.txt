🔧 TROUBLESHOOTING:
-------------------
Problem: "Cannot connect to local AI"
Rozwiązanie:
  - Sprawdź czy LM Studio/Ollama jest uruchomiony
  - Test: curl http://localhost:1234/v1/models
  - Na WSL2 użyj IP Windows: 192.168.137.1

Problem: "API key invalid"
Rozwiązanie:
  - Sprawdź czy klucz jest poprawny
  - Sprawdź czy api-base jest poprawny dla tego providera

Problem: "Import error"
Rozwiązanie:
  - pip install requests
