# System Administration Scripts (2013)

Simple bash script written in 2013 to list processes with permission restrictions workarounds.

## Origin
Written in 2013 for systems with strict permission limitations on /proc access.

More info (Polish only):
http://netcuter.wordpress.com/2013/09/11/wyswietlanie-informacji-o-wszystkich-procesach-za-pomoca-samego-basha/

## Usage
```bash
bash script.sh
```

## Features
- Process listing using only bash (no external commands like ps, top, etc.)
- Works around permission restrictions on limited systems
- Displays process information from /proc filesystem
- No external dependencies required
